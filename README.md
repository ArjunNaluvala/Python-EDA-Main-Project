# IPL Data Analysis Project

This project performs Exploratory Data Analysis (EDA) on IPL dataset to extract insights about teams, players, and match performance.


## Objective

To analyze IPL data and identify:

- Top performing teams
- Player performance
- Winning patterns
- Toss impact on match results


## Dataset

- Source: IPL Dataset
- Contains match details, teams, players, and results


## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn


## Process

1. Data Cleaning
2. Handling Missing Values
3. Data Analysis
4. Visualization
5. Insights Generation


## Key Insights

- Teams winning more matches
- Impact of toss on winning
- Top players performance
- Venue-based analysis
